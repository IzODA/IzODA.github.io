z/OS Open Data Science Platform

Welcome to z/OS Open Data Science Platform (zODSP) github pages. On our pages you will find an overall explanation of the platform, blog entries, and various documentation.

The IBM z/OS Open Data Science Platform includes solutions for interfacing with open-source data science tools including Apache Spark. One solution includes the Jupyter Notebook server. The server can run off-platform and connect to a Jupyter Kernel Gateway giving the server remote access to Jupyter kernels for example Apache Toree which is used to interact with the Spark cluster. Our most current solution (depicted in the diagram below) brings in Python 3.6 and Anaconda to z/OS. Anaconda, a package installer and environment manager, is useful when dealing with many Python libraries spanning across multiple versions.
